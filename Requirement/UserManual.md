# HealthCalculator 使用者操作手冊

歡迎使用 **HealthCalculator**！這是一個簡單易用的健康計算工具，協助您快速了解自己的身體指數，包括 BMI、BMR 與 TDEE。

## 1. 系統功能簡介

本系統提供以下三項核心計算功能：

*   **BMI (身體質量指數)**：評估您的體重是否處於健康範圍。
*   **BMR (基礎代謝率)**：計算您在靜止狀態下，維持生命所需的最低熱量。
*   **TDEE (每日總消耗熱量)**：根據您的活動量，計算維持目前體重所需的每日總熱量。

> **特色功能**：系統會自動將您的輸入資料儲存在瀏覽器中，下次開啟網頁時會自動填入，無需重複輸入。

---

## 2. 操作介面說明

### 2.1 資料輸入欄位

請依序填入以下個人資訊：

1.  **性別**：選擇「男」或「女」。
2.  **年齡**：輸入您的實歲年齡。
3.  **身高 (cm)**：輸入身高，單位為公分。
4.  **體重 (kg)**：輸入體重，單位為公斤。
5.  **活動量**：選擇您平日的活動程度：
    *   **久坐**：很少或無運動。
    *   **輕度活動**：每週運動 1-3 天。
    *   **中度活動**：每週運動 3-5 天。
    *   **高度活動**：每週運動 6-7 天。
    *   **超高度活動**：每天重度運動或勞力工作。

### 2.2 按鈕狀態與啟用條件

系統會根據您已填寫的資料，自動啟用對應的計算按鈕。若資料不完整，按鈕將呈現灰色無法點擊狀態。

| 按鈕名稱 | 啟用所需資料 |
| :--- | :--- |
| **計算 BMI** | 身高、體重 |
| **計算 BMR** | 性別、年齡、身高、體重 |
| **計算 TDEE** | 性別、年齡、身高、體重、活動量 |

---

## 3. 操作步驟

### 3.1 計算 BMI
1.  輸入 **身高** 與 **體重**。
2.  點擊藍色的 **「計算 BMI」** 按鈕。
3.  下方結果區將顯示您的 BMI 數值及健康狀態（如：過輕、正常、過重、肥胖）。

### 3.2 計算 BMR
1.  輸入 **性別**、**年齡**、**身高** 與 **體重**。
2.  點擊綠色的 **「計算 BMR」** 按鈕。
3.  下方結果區將顯示您的基礎代謝率 (kcal/day)。

### 3.3 計算 TDEE
1.  輸入 **所有欄位**（含活動量）。
2.  點擊淺藍色的 **「計算 TDEE」** 按鈕。
3.  下方結果區將顯示您的每日總消耗熱量 (kcal/day)。

## 4. 常見問題 (FAQ)

**Q: 為什麼按鈕不能按？**
A: 請檢查是否已填寫該計算所需的所有欄位。例如，計算 TDEE 必須選擇「活動量」。

**Q: 我的資料會被傳送到伺服器儲存嗎？**
A: 不會。您的個人資料僅用於當次計算，並暫存於您個人的瀏覽器 (localStorage) 中，伺服器端不會永久儲存您的隱私資訊。

**Q: 換了電腦或瀏覽器，資料還在嗎？**
A: 不在。因為資料是儲存在當前瀏覽器的暫存區，更換裝置或清除瀏覽器快取後，資料將會重置。
