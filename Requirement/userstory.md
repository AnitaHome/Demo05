# User Story: 健康計算工具

## 目標
作為一位健康管理的使用者，我希望能夠輸入自己的身體基本資訊，並根據輸入的資料計算 BMI、BMR、TDEE，方便我了解自身健康狀況。

## 使用者故事
- 當我打開應用程式時，能夠看到一個表單，讓我輸入以下身體基本資訊：
  - 性別
  - 年齡
  - 身高
  - 體重

- 在表單下方有三個按鈕：
  - **BMI**
  - **BMR**
  - **TDEE**

- 按鈕的啟用條件如下：
  - 當我輸入計算 BMI 所需的所有資訊（身高、體重）時，BMI 按鈕會變成 Enabled，否則為 Disabled。
  - 當我輸入計算 BMR 所需的所有資訊（性別、年齡、身高、體重）時，BMR 按鈕會變成 Enabled，否則為 Disabled。
  - 當我輸入計算 TDEE 所需的所有資訊（性別、年齡、身高、體重，以及活動量）時，TDEE 按鈕會變成 Enabled，否則為 Disabled。

- 當我按下已啟用的按鈕時，系統會根據我輸入的資料計算並顯示對應的結果。

- 當我按下任一按鈕（BMI、BMR、TDEE）時，系統會將目前使用者所輸入的資料儲存起來。
- 下次啟動應用程式時，系統會自動載入先前儲存的使用者資料，並顯示在表單中。

## 驗收標準
1. 使用者能夠輸入所有必要的身體資訊。
2. 三個按鈕根據輸入資訊動態啟用或停用。
3. 按下已啟用的按鈕後，能正確顯示計算結果。
4. 按下任一按鈕時，能儲存使用者輸入的資料。
5. 下次啟動時，能自動載入並顯示先前儲存的資料。
6. UI 友善且易於操作。
